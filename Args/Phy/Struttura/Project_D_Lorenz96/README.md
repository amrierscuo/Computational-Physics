# Project D · ML Emulation of Chaotic Dynamical Systems

This folder contains the reproducible material for Project D: machine-learning emulation of Lorenz-63 and Lorenz-96 chaotic dynamical systems.

## Structure

- src/: Lorenz systems and neural-network model definitions.
- scripts/: reproducible Python scripts for data generation, training, rollout evaluation, Lyapunov/FTLE diagnostics, and spectra.
- data/: small datasets used for local reproduction.
- models/: trained small MLP/CNN models.
- results/: CSV/JSON/TXT metrics and diagnostic summaries.
- figures/: selected diagnostic figures.
- notebooks/: final project notebook.
- starter_notebooks/: reference notebooks from the Lorenz-96 demo book.
- final_results/: curated final figures and reports.

## Main entry points

Run scripts in numerical order from scripts/.

The notebook notebooks/project_d_lorenz_ml_emulation.ipynb is the report-style interface.
